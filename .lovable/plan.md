
# Jai Anjaniputra Event Management Website

## Project Overview
A premium, luxurious, long-form event management website with elegant animations, rich content, and WhatsApp-driven lead generation. The design will feature a Black + Gold + White color palette with Playfair Display headings and Poppins body text.

---

## Design System

### Visual Identity
- **Primary Colors**: Deep Black (#0A0A0A), Rich Gold (#D4AF37), Pure White (#FFFFFF)
- **Accent Colors**: Soft Gold (#F5E6C8), Muted Black (#1A1A1A)
- **Typography**: Playfair Display for headings, Poppins for body text
- **Effects**: Soft shadows, gold gradients, parallax scrolling, smooth fade/slide animations
- **Dark/Light Mode**: Toggle between elegant dark theme and bright white theme
